# Plans Archive

This directory stores closed plans from `/home/konstantin/docs/plans/`.

Archive structure:

```text
archive/<year>/done/
archive/<year>/cancelled/
archive/<year>/superseded/
```

Rules:

- Archived plans are historical audit trail, not active TODO.
- Do not treat archived plans as current source of truth without checking current docs/config/tools.
- Durable outcomes should be promoted to `../infrastructure.md`, `../runbooks.md`, relevant skills, or holographic memory.
- Do not delete archived plans unless they contain secrets, severe misinformation, or obvious trash.
